    ANSWER: ap Packs the contents of array into a binary sequence according to the directives in aTemplateString. If the count is an asterisk (*STAR), all remaining array elements will be converted. Any of the directives is still may be followed by an underscore (_) to use the underlying platform's native size for the specified type; otherwise, they use a platformindependent size. Spaces are ignored in the template string. ( See templating Table below )
