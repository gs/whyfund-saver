    An object can contain data, including references to other objects.

    true

    An object's methods can call/run other methods/functions.

    true

      

    methods that end with an exclamation mark (also called a "bang") modify the object.

    true

      

    The method upcase! changes the letters of a String to uppercase.

    true

      

    Methods that do not end in an exclamation point return data, but do not modify the object. For example, 'downcase!' modifies a String object by making all of the letters lowercase. However, 'downcase' returns a lowercase copy of the String, but the original string remains the same.

    true
