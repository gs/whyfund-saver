    [BasicObject]

    It is the parent class of all classes in Ruby. It's an explicit blank class.

      

    [Object]

    It is the default root of all Ruby objects. It inherits from BasicObject which allows creating alternate object hierarchies.

      

    [ARGF]

    It is a stream designed for use in scripts that process files given as command-line arguments or passed in via STDIN. The arguments passed to your script are stored an Array, one argument per element.

      

    [Array]

    It is an ordered, integer-indexed collection of any object.

      

    [Binding]

    Objects of this class encapsulate the execution context at some particular place in the code and retain this context for future use.

      

    [Continuation]

    This object is generated by Kernel#callcc, after having +require+d continuation. It holds a return address and execution context, allowing a nonlocal return to the end of the callcc block from anywhere within a program.

      

    [Data]

    This is a recommended base class for C extensions.

      

    [Dir]

    Objects of this class are directory streams representing directories in the underlying file system.

      

    [Encoding]

    An instance of this class represents a character encoding usable in Ruby.

      

    [Enumerator]

    A class which allows both internal and external iteration.

      

    [ENV]

    It is a hash-like accessor for environment variables.

      

    [Exception]

    Descendants of this class are used to communicate between Kernel#raise and rescue statements in begin ... end blocks.

      

    [FalseClass]

    The global value false is the only instance of this class and represents a logically false value in boolean expressions.

      

    [Fiber]

    This is a primitive for implementing light weight cooperative concurrency in Ruby. Basically it is a means of creating code blocks that can be paused and resumed, much like threads.

      

    [File::Stat]

    Objects of this class encapsulate common status information for File objects.

      

    [Hash]

    It is a dictionary-like collection of unique keys and their values. (Also called associative arrays)

      

    [IO]

    This class is the basis for all input and output in Ruby.

      

    [File]

    This is an abstraction of any file object accessible by the program and is closely associated with class IO.

      

    [MatchData]

    This is the type of the special variable $~, and is the type of the object returned by Regexp#match and Regexp.last_match.

      

    [Module]

    This is a collection of methods and constants.

      

    [Class]

    Classes in Ruby are first-class objects---each is an instance of this class

      

    [Mutex]

    It implements a simple semaphore that can be used to coordinate access to shared data from multiple concurrent threads.

      

    [Proc]

    This object is blocks of code that has been bound to a set of local variables. Once bound, the code may be called in different contexts and still access those variables.

      

    [Process::Status]

    This object encapsulates the information on the status of a running or terminated system process.

      

    [Random]

    This provides an interface to Ruby's pseudo-random number generator, or PRNG.

      

    [Range]

    This represents an interval—a set of values with a beginning and an end.

      

    [Regexp]

    This holds a pattern which describe the contents of a string, which is used to match a pattern against strings.

      

    [String]

    This object holds and manipulates an arbitrary sequence of bytes, typically representing characters.

      

    [Struct]

    This is a convenient way to bundle a number of attributes together, using accessor methods, without having to write an explicit class.

      

    [Symbol]

    This object represents a name or some string. The same object will be created for a given name or string for the duration of a program's execution, regardless of the context or meaning of that name.

      

    [ThreadGroup]

    This provides a means of keeping track of a number of threads as a group.

      

    [TrueClass]

    The global value true is the only instance of this class and represents a logically true value in boolean expressions.

      

    [Time]

    It is an abstraction of dates and times.

      

    [NilClass]

    The class of the singleton object nil.

      

    [Numeric]

    The class whose descendants include Integer, Float, and Rational

      

    [Complex]

    This class of number can be represented as a paired real number with imaginary unit.
    (e.g. 1+2i)

      

    [Float]

    These objects represent inexact real numbers using the native architecture's double-precision floating point representation.

      

    [Integer]

    This class is the basis for the two concrete classes that hold whole numbers, Bignum and Fixnum.

      

    [Bignum]

    This hold integers outside the range of Fixnum.

      

    [Fixnum]

    This holds Integer values that can be represented in a native machine word (minus 1 bit).

      

    [Rational]

    This class of number can be represented as a paired integer number; a/b (b>0). Where a is numerator and b is denominator.

    [UnboundMethod]
