    What does this command mean? git remote add origin git@github.com:<username>/first_app.git

    "git remote add" means you want to create a remote repository. "origin" is the alias or name for that remote that you will use to refer to it. The last argument is the actual address of the remote repository.

    What does "git push -u origin master" mean?

    It means push the "master" commit that you have made locally to the repo called "origin". The "master" commit is usually the last one you have made locally. The -u flag means you are setting "origin" as your default upstream repo. :)
