    single-quoted string

    string that does not allow for interpolation or backslashing except \' and \\

    (Not faster than double-quoted strings.)

      

    double-quoted string

    string that allows for interpolation and backslashing

      

    interpolate string

    "my variable: #{myVariable}"

      

    special character-delimited strings

    %(this string can have (unescaped) parentheses if in balanced pairs)
    %~the tilde (\~) will always need to be escaped~

      

    modifiers for special character-delimited strings

    %q[ ] -- Non-interpolated String (except for \\ \[ and \])
    %Q[ ] -- Interpolated String (default)
    %r[ ] -- Interpolated Regexp (flags can appear after the closing delimiter)

    basic auth

    uri = URI('http://example.com/index.html?key=value')

    req = Net::HTTP::Get.new(uri)
    req.basic_auth 'user', 'pass'

    res = Net::HTTP.start(uri.hostname, uri.port) {|http|
    http.request(req)
    }

      

    make single GET request by URI

    uri = URI('http://example.com/index.html?count=10')
    Net::HTTP.get(uri) # => String

      

    make single parameterized GET request by URI

    uri = URI('http://example.com/index.html')
    params = { :limit => 10, :page => 3 }
    uri.query = URI.encode_www_form(params)

    res = Net::HTTP.get_response(uri)
    puts res.body if res.is_a?(Net::HTTPSuccess)

      

    make GET request over persistent connection

    uri = URI('http://example.com/some_path?query=string')

    Net::HTTP.start(uri.host, uri.port) do |http|
    request = Net::HTTP::Get.new uri

    response = http.request request # Net::HTTPResponse object
    end
