    tell me about format strings & their syntax

    format strings are strings that take multiple variables assigned differently in different instances of that that string being printed

    syntax:

    format_string = "some text %{variable_one}, %{variable_two}, more text %{variable_three}"

    puts format_string % {variable_one: 1, variable_two: 2, variable_three: 3}

    output: some text 1, 2, more text 3
